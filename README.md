# Habitile プライバシーポリシー

最終更新日：2026年4月30日

## 1. はじめに

Habitile（以下「本アプリ」）は、猿渡俊也（以下「開発者」）が開発・提供する習慣管理アプリです。本プライバシーポリシーは、本アプリにおける個人情報の取り扱いについて説明します。

## 2. 収集する情報

### ユーザーが入力する情報
- 習慣の名前、目標、記録データ
- これらのデータはお使いのデバイス内にのみ保存され、開発者のサーバーには送信されません。

### 自動的に収集される情報
- 広告配信のため、Google AdMobによる広告識別子（IDFA）等の情報が収集される場合があります。

## 3. 情報の利用目的
- アプリ機能の提供（習慣の記録・管理）
- 広告の配信（無料版のみ）

## 4. 第三者への情報提供

開発者は、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。
- ユーザーの同意がある場合
- 法令に基づく場合

### 広告サービス
本アプリはGoogle AdMobを使用しています。
https://policies.google.com/privacy

## 5. データの保存

ユーザーが入力した習慣データは、デバイス内にのみ保存されます。開発者はこれらのデータにアクセスすることはできません。

## 6. お子様のプライバシー

本アプリは13歳未満のお子様を対象としていません。

## 7. プライバシーポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。変更後のポリシーはこのページに掲載されます。
